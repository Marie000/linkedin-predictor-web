This application was built as a machine learning experiment by:

Marie Pelletier

This is not meant to be used as an accurate predictor of salary. It is limited by the data that was used, which is US-based, from 2023. It also does not take into account regions or fluctuations in the market over time. The dataset included over 33,000 job postings, but only 13,000 had salary information.

For all you statistic nerds, the r-squared score of the model was around 0.57. It might have been better if I had more data to work with. Or maybe there is just so much we can infer from a description alone.
